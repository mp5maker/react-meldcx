# MeldCX App

### Installation Used

npx create-react-app . --template=typescript

### Commands

- yarn build
- yarn start
- yarnt test

### Packages Used

- lodash
- sass
- npm install @mui/material
- formik
- yup
- axios
- react-router-dom

### Code Format

- husky
- eslint
- prettier

### Enable Githooks

npx husky install
npx husky add .husky/pre-commit "npx --no-install lint-staged"
